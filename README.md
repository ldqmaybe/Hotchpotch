# 优了个秀
- **GitHub地址：** https://github.com/ldqmaybe/hotch-client

《优了个秀》是基于MVP+RxJava2+Retrofit2+Glide等架构实现的一个资讯类应用。
采用了 [《干货集中营》](http://gank.io/api) 的分类数据接口的all和福利分类
